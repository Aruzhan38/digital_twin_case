# Digital Twin Dashboard for Locomotive Monitoring
# Problem

В современных железнодорожных системах операторы сталкиваются с огромным потоком телеметрических данных: скорость, температура, давление, уровень топлива, электрические параметры и сигналы ошибок.

# Эти данные поступают в реальном времени, но:

они разрознены
сложны для интерпретации
не дают мгновенного понимания состояния системы

# В результате:

ошибки в принятии решений
риск отказов
снижение эффективности эксплуатации

## Run the backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Run the frontend

```bash
cd frontend
npm install
npm run dev
```
